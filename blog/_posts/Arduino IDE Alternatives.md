**Alternatives to the Arduino IDE**

If you are not particularly fond of the Arduino IDE - and lets face it, who could be - there are several alternatives that you might like to check out if you develop on MacOS.

* Use xCode the development IDE used to create Mac and IOS apps.<br>
	There are a ton of xCode templates around such as [xavr](https://github.com/jawher/xavr). [Here is a blog post](https://jawher.me/2014/03/21/using-xcode-avr-c/) about using xavr. Another one is [embedXcode](https://embedxcode.weebly.com) which I have used.
* Use PlatformIO
